OpenCV를 활용한 영상 출력   
===  

>  기능  
- 's'를 누르면 출력되는 컬러영상의 흑백 버전이 저장  
- 'i'를 누르면 두 영상의 차이값으로 표현된 차영상이 생성되어 출력
